# UVERworld Setlist Archive

UVERworldのライブセットリストを記録・閲覧するための非公式Webアプリケーションです。
「過去のライブのセットリストを振り返りたい」「あの時の曲順を知りたい」というファンの声に応えるために作成されました。

## 機能
- **ライブ一覧**: 過去のライブを日付順に表示
- **フィルタリング**: 開催年やキーワードで絞り込み
- **セットリスト詳細**: 曲順やアンコール情報を閲覧
- **My Page**: 参戦記録と聴いた楽曲の統計
- **修正リクエスト**: 誤りの報告機能

## 技術スタック
- Frontend: React (Vite)
- Backend: Node.js (Express)
- Database: PostgreSQL
