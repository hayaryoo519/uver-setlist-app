# コミット＆プッシュ

// turbo-all

1. `git status` で変更ファイルを確認する

2. `git diff --staged` および `git diff` で変更内容の差分を取得する

3. 変更内容を分析し、以下のフォーマットで**日本語のコミットメッセージ**を自動生成する
   - プレフィックス: `feat:` / `fix:` / `refactor:` / `docs:` / `style:` / `chore:` のいずれか
   - 本文: 変更内容を簡潔に日本語で説明
   - 例: `fix: LATEST LIVEの表示条件を今日以前に変更`

4. 全変更ファイルを `git add .` でステージングする

5. 生成したコミットメッセージで `git commit` を実行する

6. 現在のブランチに `git push` を実行する
   - ブランチ名は `git branch --show-current` で取得すること

7. 完了をユーザーに報告する（コミットハッシュ、ブランチ名、変更ファイル一覧を含める）
